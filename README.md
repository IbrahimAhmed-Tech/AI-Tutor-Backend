# AI Tutor - Backend

The backend for AI Tutor, built with **Node.js** and **Express**, handling:

✅ Speech-to-text conversion via Whisper AI  
✅ AI response generation via ChatGPT (LLM)  
✅ Text-to-speech conversion via Google TTS  
✅ Database logging of user inputs & outputs using Supabase  

---

## 🔧 Tech Stack

- Node.js  
- Express.js  
- Supabase (PostgreSQL database)  
- Whisper AI (speech-to-text)  
- OpenAI ChatGPT (LLM for replies)  
- Google Cloud TTS (text-to-speech)
